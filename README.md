# Arabic AI vs Human Text Detection

## Overview
This project focuses on detecting whether an Arabic text was written by a human or generated by AI.  
It applies Natural Language Processing techniques, classical machine learning models, and deep learning models to compare performance and identify the best approach for Arabic text classification.

## Dataset
The dataset used in this project is included in the `data/` folder.
- Human-written text
- AI-generated text

Dataset summary:
- Total samples: 1,494
- Human-written samples: 735
- AI-generated samples: 759

Deep Learning models:

- Feedforward Neural Network
- LSTM
- BiLSTM

## Best Result
The best-performing model was a tuned Support Vector Machine using TF-IDF features.

Performance:

- Accuracy: 86.62%
- F1-score: 86.58%

## Key Takeaways
- TF-IDF with SVM achieved the strongest overall performance.
- Classical machine learning performed very well on this small-to-medium Arabic NLP dataset.
- Deep learning models required more data to outperform classical approaches.
- Stopword handling affected the performance of classical and deep learning models differently.
